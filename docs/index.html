So abonnierst du den Kalender

iPhone / iPad: Tippe auf „Abonnieren". iOS öffnet einen Dialog, mit dem du den Kalender direkt hinzufügen kannst.

Google Kalender: URL kopieren → calendar.google.com → links auf „+ Weitere Kalender" → „Per URL" → URL einfügen. Google aktualisiert abonnierte Kalender etwa alle 12–24 Stunden.

Apple Kalender (Mac): Datei → Neues Kalenderabo → URL einfügen. Aktualisierungs-Intervall kannst du auf 5 Minuten stellen.

Outlook: Kalender hinzufügen → Aus dem Internet abonnieren → URL einfügen.

Wie funktioniert die Aktualisierung?

Einmal täglich wird der HTML-Stundenplan der EHB automatisch abgerufen und in ICS-Dateien umgewandelt. Wenn sich Termine ändern (neue Räume, verschobene Stunden, entfallene Termine), übernimmt dein Kalender-App die Änderungen automatisch beim nächsten Polling.

Events haben stabile IDs — Updates überschreiben das alte Event, statt Duplikate zu erzeugen.

Warum zwei Feeds?

Es gibt zwei unabhängige Gruppen-Einteilungen: Großgruppen A–D und Kleingruppen 1a–3b. Der Großgruppen-Feed enthält bereits alle Plenum-Termine (für alle Studierenden), dein Kleingruppen-Feed nur die zusätzlichen Termine deiner Kleingruppe. So hast du keine Duplikate.
